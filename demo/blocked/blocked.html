Blocked Host — CSP Demo

This origin is NOT in allowed_origins.
The widget bubble below should NOT appear. Open DevTools → Console to see the Content-Security-Policy: frame-ancestors violation.

This page is served from http://localhost:8090. The widget's widget.js response carries: 

Content-Security-Policy: frame-ancestors 'self' http://localhost:8080

Since http://localhost:8090 is not in the frame-ancestors list, the browser refuses to load the widget into this page's context and logs a CSP violation in the console.

Compare with the allowed host at http://localhost:8080 where the bubble appears.
