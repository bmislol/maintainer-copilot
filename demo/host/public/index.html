Acme OSS Docs Demo Host

This page simulates a third-party documentation site that has embedded the Maintainer's Copilot widget. The chat bubble in the bottom-right corner is loaded from the api at http://localhost:8000.

This host runs on http://localhost:8080. For the widget to load, http://localhost:8080 must be in the widget's allowed_origins list (set in the Streamlit admin → Widget Configuration page). If the origin is not allowed, the browser will block the embed and you will see a Content-Security-Policy violation in the console.

Sample Documentation

Installation:

pip install acme-library

Quick start:

from acme import Client
client = Client(api_key="...")
result = client.run(query="hello world")
